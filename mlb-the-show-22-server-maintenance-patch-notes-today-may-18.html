MLB The Show 22 Server Maintenance &amp; Patch Notes Today (May 18)

Today, Sony’s San Diego Studios will roll out MLB The Show 22 server update, addressing a backend issue with the playoffs of Mini Seasons, however, there will be maintenance and downtime before the deployment of the update.

If you have already set aside a time for MLB The Show 22 for today, you might need to reschedule your plans as the game will go under maintenance soon.

Earlier today, the official Twitter account of MLB The Show 22 shared a new tweet, revealing upcoming server maintenance that will address a certain connectivity issue in the playoffs games of Mini Seasons. Down below, you can check out the full patch notes as well as the start time of the maintenance.

MLB The Show 22 Server Maintenance Schedule Today (May 18)

According to San Diego Studios, MLB The Show 22 servers will go under maintenance on May 18 at 4 am PT/7 am ET/11 am UTC/12 pm BST/1 pm CEST/10 pm AEDT. Unfortunately, we don’t know how long the maintenance would take, but you will have no access to the online features of the game during the downtime.

MLB The Show 22 Server Update Patch Notes

  • This update will have a fix for the “unhandled server exception” in the playoffs for Mini Seasons.

This is only a server update, which means you don’t need to download any new data after the conclusion of the maintenance. This improvement will be implemented across all platforms. Also, make sure to finish your online games before the beginning of maintenance or your progress will be lost.

MLB The Show 22 is now available on PS4, PS5, Xbox One, Xbox Series X, Xbox Series S, and Nintendo Switch. It’s worth noting that you can play it for free via an Xbox Game Pass subscription.
